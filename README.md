## Featured projects

### [Aircraft Aerodynamic Analysis Using XFLR5](projects/aircraft-aerodynamic-analysis-xflr5/)
Fixed-wing aerodynamic and stability analysis using XFLR5, including lift, drag, pitching moment and aircraft geometry.

### [Lightweight Bicycle Crank Optimisation](projects/bicycle-crank-optimisation/)
CAD redesign and finite element validation of a lightweight bicycle crank, reducing mass from approximately 727 g to 297 g.

### [Bench Vice CAD Assembly](projects/bench-vice-cad/)
Parametric component modelling, assembly construction, exploded views and engineering drawings for a bench vice.

### [FEA Stress Concentration Analysis](projects/fea-stress-concentration-analysis/)
ANSYS mesh-convergence study investigating nominal stress, stress concentrations, singularities and fillet effects.

### [PID Control of a Mass-Spring-Damper System](projects/mass-spring-damper-pid-control/)
System identification and proportional, derivative and integral control analysis using MATLAB and experimental data.

### [Rotational Position and Speed Control](projects/rotational-position-speed-control/)
Modelling and feedback-control analysis of a torsional mechanical system using transfer functions and MATLAB.
